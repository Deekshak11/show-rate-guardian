# Security note

This public repository is a **portfolio snapshot** of work built by Deekshak SS.

## What is intentionally omitted

- API keys, OAuth tokens, app passwords
- Production lead lists / client PII
- Live server credentials and auth passwords
- Private knowledge vaults

## What remains private

Full historical / runtime backups (e.g. `Agency_OS`, `Agentic-OS`) stay **private** on GitHub. They may contain operational history not safe for public distribution.

## If you find a secret

Contact the owner immediately. Rotate the credential. Do not reuse any example placeholders.
