# Discovery Questionnaire

Collect the profile in compact groups. Keep questions concrete.

## 1. Operator and Goal

- Client or stack name
- Primary timezone
- Main objective: higher kept-call rate, lower no-show rate, smoother reschedules, cleaner CRM reporting
- Success metric and target lift

## 2. Stack

- CRM
- Scheduler
- Calendar source
- SMS provider
- Email provider
- Telephony provider, if any

## 3. Channels and Policies

- Primary reminder channels
- Fallback channels
- Reminder cadence
- Confirmation requirement
- Reschedule cutoff window
- Working hours or routing windows
- No-show grace period

## 4. Reporting and Escalation

- Required metrics
- Where show/no-show should be recorded
- Human escalation path for failures or edge cases

## 5. Deployment Constraints

- Repo root on host
- Hermes runtime location
- Web tools available or not
- Whether the first rollout should be read-only before writes
